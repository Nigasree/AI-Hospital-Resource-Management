
# AI Hospital Resource Management System

The AI Hospital Resource Management System is an intelligent platform designed to monitor and manage hospital resources such as beds, ICU units, ventilators, and medical staff. It uses data analytics and AI to track resource availability in real time and assist hospital administrators in making better decisions. The system provides a dashboard that displays current resource status and helps predict future demand, improving hospital efficiency and patient care.


## How It Works


1.The system collects hospital resource data such as beds, ICU units, ventilators, and patient information.

2.he data is processed using Python and data analysis libraries.

3.The AI model analyzes the data to understand resource usage patterns.

4.A dashboard displays the current availability of hospital resources in real time.

5.The system can predict future demand and help hospital staff manage resources more efficiently.
## Tech Stack
**Programming Language**: Python

**Dashboard**: Streamlit

**Data Processing**: Pandas, NumPy

**Data Visualization**: Matplotlib, Plotly

**Machine Learning**: Scikit-learn

**Dataset**: MySql

**Development Environment**: VS Code
